    7. How it works
    
          
    
            1. Data Input: The user uploads a CSV file containing historical
            electricity consumption with DateTime and Consumption_KWh columns.

    
            2. Pre-processing: The data is cleaned, sorted by time and scaled
            using a saved scaler. Only the most recent rows are used to keep the model fast.

    
            3. LSTM Model: A trained LSTM (Long Short-Term Memory) neural
            network learns temporal patterns from past consumption values and predicts
            future load step-by-step for the selected horizon (hours/days/months/years).

    
            4. Hybrid Profiles: The same forecast is interpreted using
            household, apartment or industrial reference levels to generate profile-aware
            severity and cost insights.

    
            5. Intelligent Safety Layer: Based on predicted usage level,
            peak hours, time-of-day, weather and profile severity, the system generates
            5-point safety and energy saving suggestions.
